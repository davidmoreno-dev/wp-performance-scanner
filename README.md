# WP Performance Scanner

Micro-SaaS para analizar webs WordPress y generar informes automáticos de rendimiento y calidad frontend.

## Stack
- Next.js
- TypeScript
- Supabase
- PostgreSQL
- Playwright
- BullMQ
- Redis

## Apps
- `apps/web`: app web y API
- `apps/worker`: worker de procesamiento
- `packages/shared`: tipos y utilidades compartidas

## Scripts
- `pnpm dev:web`
- `pnpm dev:worker`
- `pnpm build`
- `pnpm lint`
- `pnpm typecheck`
